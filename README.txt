=== README ===

* build with `make all`
* run with 'make run' (obv)
* dependencies:
    cmake
    rsync
    clang (15+)
* if things are broken, ensure that git submodules have been cloned recursively
  (for missing SDL[1], SDL_Image[1], cimgui, or sokol, ...)

[1] -> SDL2 can be found under the official git repo under releases 
        just scroll down a bit
